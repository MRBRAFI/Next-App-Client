# CourseVerse - Next.js Course Management Platform

[Live Demo](https://next-app-client-seven.vercel.app/)

---

## **Project Description**

**CourseVerse** is a modern, full-stack web application built with **Next.js**, **Firebase Authentication**, and **MongoDB (Express server)**.  
It allows users to browse courses, view course details, and manage their own courses. Authenticated users can add, view, and delete their own courses. The app features:

- User Authentication via **Firebase Auth** (email/password & Google login)
- Private routes to protect authenticated pages
- CRUD operations for courses
- Dynamic course detail pages
- Responsive and modern UI with Tailwind CSS
- SweetAlert integration for user-friendly confirmations
- Search functionality for courses
